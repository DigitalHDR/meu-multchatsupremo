Como configurar no OBS

  1. Copie o arquivo .env.example para .env e preencha seus canais.
  2. Execute npm start na pasta do projeto.
  3. No OBS: Fontes → + → Navegador (Browser Source).
  4. Cole a URL acima. Defina largura 400 e altura 600 (ajuste como quiser).
  5. Marque "Desativar origem quando não visível" para economizar recursos.
  6. O fundo é transparente — o chat aparece sobre sua cena.
