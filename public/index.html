Configuração dos canais

Use o canal OU o ID do vídeo. Se os dois estiverem preenchidos, o ID do vídeo tem prioridade.
Escolha uma das portas fixas. Verde = livre · Vermelho = em uso por outro programa

Após salvar, reinicie o servidor (feche e abra o iniciar.bat) para aplicar as mudanças.
